## 🛠️ How to Use

1. Clone this repo:
   ```bash
   git clone https://github.com/vnair85/infra-security-gpt.git
   cd infra-security-gpt
   ```

2. Install Streamlit:
   ```bash
   pip install streamlit
   ```

3. Run the app:
   ```bash
   streamlit run infra_security_gpt.py
   ```

4. Visit `http://localhost:8501` to use the tool.
